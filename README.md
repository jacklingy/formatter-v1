# 文档格式一键转换器 V1.1 - 使用说明

## 项目概述

这是一个专业的文档格式转换工具，支持将 Markdown 文件转换为格式化的 Word 文档，以及对现有 Word 文档进行批量格式化处理。

**版本**: V1.1  
**更新内容**:
- ✅ 修复了 Word 格式化时的 `bulletListPr` 属性错误
- ✅ 改进了列表段落检测逻辑
- ✅ 增强了兼容性和稳定性

## 功能特性

### 核心功能
1. **Markdown 转 Word**：将.md 文件转换为带有专业格式的.docx 文档
2. **Word 格式化**：对现有.doc/.docx 文件进行批量格式化（字体、段落、标题、间距等）
3. **自定义配置**：通过 YAML 配置文件灵活控制各层级标题和段落的格式

### 支持的格式元素
- 6 级标题样式（可自定义字体、字号、颜色、对齐方式等）
- 普通段落（行间距、首行缩进、段前段后间距）
- 有序列表和无序列表
- 代码块
- 表格
- 页边距设置
