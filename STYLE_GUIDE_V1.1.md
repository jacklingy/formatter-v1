## 🔧 技术实现细节

### 1. 标题样式配置结构

```yaml
heading_styles:
  heading1:                    # 一级标题
    font_name: 宋体            # 字体名称
    font_size: 18              # 字号（磅值）
    bold: true                 # 是否加粗
    alignment: center          # 对齐方式：left/center/right/justify
    space_before: 240          # 段前间距（twips，240=12pt）
    space_after: 120           # 段后间距（twips，120=6pt）
    color: '000000'            # 文字颜色（十六进制）
```

**字号对照表**：
| 中文字号 | 磅值(pt) | twips值 |
|---------|---------|---------|
| 初号 | 42 | 840 |
| 小初 | 36 | 720 |
| 一号 | 26 | 520 |
| 小一 | 24 | 480 |
| 二号 | 22 | 440 |
| 小二 | 18 | 360 |
| 三号 | 16 | 320 |
| 小三 | 15 | 300 |
| 四号 | 14 | 280 |
| 小四 | 12 | 240 |
| 五号 | 10.5 | 210 |
| 小五 | 9 | 180 |

### 2. 段落格式配置

```yaml
paragraph_style:
  line_spacing: 1.5            # 行距倍数
  first_line_indent: 480       # 首行缩进（twips）
                              # 480 twips ≈ 24pt ≈ 2个字符（小四号）
  space_before: 0              # 段前间距
  space_after: 0               # 段后间距
```

**首行缩进计算公式**：
```
缩进(twips) = 字号(pt) × 20 × 缩进字符数
示例：小四号(12pt) × 20 × 2字符 = 480 twips
```

### 3. 表格样式配置（新增功能）⭐

```yaml
table_style:
  border_color: '000000'       # 边框颜色
  border_width: 1              # 边框宽度（pt）
  cell_padding: 100            # 单元格内边距（twips）
  
  header:                      # 表头样式（第一行）
    font_name: 宋体
    font_size: 12              # 小四
    bold: true                 # 加粗
    background_color: [215, 215, 215]  # 背景色RGB值
    text_alignment: center     # 居中对齐
  
  body:                        # 表体样式（第二行起）
    font_name: 宋体
    font_size: 12              # 小四
    bold: false                # 不加粗
    background_color: [255, 255, 255]  # 白色背景
    text_alignment: center     # 居中对齐
```

**背景色设置说明**：
- 使用 RGB 数组格式：`[R, G, B]`
- 取值范围：0-255
- 常用颜色：
  - 白色：`[255, 255, 255]`
  - 黑色：`[0, 0, 0]`
  - 浅灰：`[215, 215, 215]`
  - 浅蓝：`[220, 235, 250]`
  - 浅绿：`[220, 245, 220]`
  - 浅黄：`[255, 255, 200]`

## 📝 使用方法

### 方式一：通过 GUI 编辑（推荐）

1. 启动程序 `文档格式一键转换器 V1.1.exe`
2. 点击右下角"**格式设置**"按钮
3. 系统会自动打开 `format_config.yaml` 文件
4. 使用文本编辑器修改配置
5. 保存后重新运行程序即可生效

### 方式二：直接编辑配置文件

1. 找到项目目录中的 `format_config.yaml`
2. 使用任意文本编辑器打开（推荐 VS Code、Notepad++ 等）
3. 修改需要的配置项
4. 保存文件（确保编码为 UTF-8）

### 验证配置是否正确

```bash
# 运行验证脚本
python validate_config.py
```

输出示例：
```

  heading1: 宋体, 18pt, 加粗, 对齐:center
  heading2: 黑体, 12pt, 加粗, 对齐:left
  ...

✅ 所有配置项验证通过！
```

## ⚙️ 自定义配置指南

### 修改字体

支持的中文字体示例：
- **宋体**: SimSun（Windows 默认）
- **黑体**: SimHei
- **楷体**: KaiTi / 楷体_GB2312
- **仿宋**: FangSong / 仿宋_GB2312
- **微软雅黑**: Microsoft YaHei
- **方正**: FZShuTi、FZKaiTi 等

### 修改字号

常用文档字号建议：

| 场景 | 推荐字号 | 说明 |
|------|---------|------|
| 论文主标题 | 小二 (18pt) 或 三号 (16pt) | 显眼醒目 |
| 章节标题 | 四号 (14pt) 或 小三 (15pt) | 层次分明 |
| 小节标题 | 小四 (12pt) | 清晰易读 |
| 正文 | 小四 (12pt) 或 五号 (10.5pt) | 舒适阅读 |
| 注释/图注 | 五号 (10.5pt) 或 小五 (9pt) | 辅助说明 |

### 修改对齐方式

可选值：
- `left`: 左对齐（中文正文标准）
- `center`: 居中（标题、表格）
- `right`: 右对齐（日期、署名等）
- `justify`: 两端对齐（英文排版常用）

### 修改段间距

```yaml
space_before: 240    # 段前空一行（12pt）
space_after: 120     # 段后半行（6pt）
```

**单位换算**：
- 1 pt (磅) = 20 twips
- 1 英寸 = 72 pt = 1440 twips
- 1 cm ≈ 567 twips

## ❓ 常见问题

### Q1: 为什么有些字体显示不出来？

**A**: 目标电脑需要安装对应的字体。常见解决方案：
- Windows: 字体通常已内置宋体、黑体、楷体、仿宋
- Linux: 需要安装 `fonts-wqy-*` 或 `cjkuni-*` 字体包
- 建议：优先使用系统自带字体以确保兼容性

### Q2: 如何让所有标题都加粗？

**A**: 在配置中将每个标题的 `bold` 都设为 `true`:

```yaml
heading_styles:
  heading1:
    bold: true
  heading2:
    bold: true
  # ... 其他标题同样设置
```

### Q3: 如何修改表格的边框样式？

**A**: 修改 `table_style` 部分：

```yaml
table_style:
  border_color: 'FF0000'    # 红色边框
  border_width: 2           # 边框宽度 2pt
  cell_padding: 150         # 更大的内边距
```

### Q4: 背景色不生效怎么办？

**A**: 检查以下几点：
1. 确保 `background_color` 格式正确：`[R, G, B]`
2. RGB 值在 0-255 范围内
3. 保存文件时使用 UTF-8 编码
4. 重新启动程序使配置生效

### Q5: 如何恢复默认配置？

**A**: 删除当前的 `format_config.yaml`，程序会自动生成默认模板。

## 🔍 高级技巧

### 1. 创建多个配置文件

```bash
# 学术论文配置
cp format_config.yaml format_config_academic.yaml

# 个人笔记配置

```

### 2. 批量替换字体

如果需要将整个文档的字体从 A 改为 B：

```python
# 在 Python 中批量处理
import yaml

with open('format_config.yaml', 'r', encoding='utf-8') as f:
    config = yaml.safe_load(f)

# 替换所有字体
old_font = '宋体'
new_font = '微软雅黑'

config['font_settings']['default_font'] = new_font
for heading in config['heading_styles'].values():
    if heading['font_name'] == old_font:
        heading['font_name'] = new_font

with open('format_config.yaml', 'w', encoding='utf-8') as f:
    yaml.dump(config, f, allow_unicode=True, default_flow_style=False)
```
